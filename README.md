It's a sample view style,just one file ,you can modify it to your project.


## Install

In your project build.gradle:

```sh
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

In your needed module
```sh
	dependencies {
		implementation 'com.github.ArronYee:GearView:0.0.7'
	}
```
